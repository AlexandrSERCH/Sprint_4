# Sprint_4

1. Добавление новой книги (add_new_book)
- Успешное добавление новой книги 
- Валидация на добавление дубликата
- Валидация на длину (от 0 до 40 включительно)

2. Установление книги жанра (set_book_genre)
- Успешное установление книге жанра
- Валидация на наличие книги в словаре
- Валидация на наличие жанра в списке

3. Получение жанра книги по её имени (get_book_genre)
- Успешное получение жанра существующей книги по её имени
- Отсутствие ошибок при запросе несуществующей книги

4. Вывод списка книг с определённым жанром (get_books_with_specific_genre)
- Успешное получение списка книг с определённым жанром
- Валидация на пустой словарь книг
- Валидация на несуществующий жанр книги

5. Получение словаря книг с жанрами (get_books_genre)
- Успешное получение словаря книг с жанрами

6. Возвращение книг, подходящие детям (get_books_for_children)
- Успешное возвращение книг, с валидацией на запрещенные жанры

7. Добавление книги в Избранное (add_book_in_favorites)
- Успешное добавление книги в Избранное
- Валидация на добавление дубликата

8. Удаление книги из Избранного (delete_book_from_favorites)
- Успешное удаление книги из Избранного

9. Получение Избранных книг(get_list_of_favorites_books)
- Успешное получение Избранных книг
